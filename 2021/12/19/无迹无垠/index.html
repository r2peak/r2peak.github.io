不辞而别或许是才是世间常态，有些人永远留在了昨天。

昨晚萌生强烈的念头：搭建个人博客

今天一早上算是初步实现了，一系列小bug比如在配置文件里面需要加空格，以及必须所有的都符合语法规范，才能部署deploy上GitHub，明天看看gitee 那边实名认证通过没，还是得部署到国内的上面，速度，稳定性才能得到一个保证。

Github 连接太不稳定了，经常hexo d 不上去

三部曲:
hexo clean
hexo g 生成generate
hexo d 部署

从昨晚的念头，到今晚的实现复盘：强烈的渴望+坚定不移的信念 = success

一步一步慢慢走，才更快！

还剩368天，得只争朝夕啊！

时间不会回头，错过就是错过了。
